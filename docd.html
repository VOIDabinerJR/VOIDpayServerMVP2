Integração do Botão de Pagamento

Esta documentação descreve como integrar um botão de pagamento na sua página web usando o SDK fornecido. O botão permitirá que os usuários realizem pagamentos de maneira segura e eficiente.

Estrutura HTML

Código HTML

<div class="product-item">
    <div>
        <img class="image-item" style="width:45px;" src="..." alt="Imagem do Produto 5">
    </div>
    <div class="name-item">Produto 5</div>
    <div class="quantity-item">2</div>
    <div class="price-item">2200</div>
    <div class="id-item">005</div>
</div>

<div id="void-button-container" buttonToken="VOID-04fce89f-0952-4736-8c35-c93c0e503809"></div>
<div id="buttonid" name="VOID-04fce89f-0952-4736-8c35-c93c0e503809"></div>
<script src="http://localhost:3000/sdk/js?clientId=49db28ad-8ce0-4370-aefd-3f8636b6081b&acept=false"></script>

Passos para Integração

1. Adicione o Código HTML

Adicione o código HTML fornecido ao corpo da sua página onde deseja que o botão de pagamento apareça.

2. Configure o Token do Botão

Certifique-se de substituir o valor do atributo buttonToken e name no elemento div com o token do botão fornecido pelo serviço de pagamento. Este token identifica sua conta de pagamento e deve ser único para cada botão.

3. Inclua o SDK

Inclua o SDK do serviço de pagamento na sua página, conforme mostrado no exemplo de código acima. O SDK é responsável por renderizar o botão de pagamento e gerenciar a comunicação com o serviço de pagamento.

<script src="http://localhost:3000/sdk/js?clientId=49db28ad-8ce0-4370-aefd-3f8636b6081b&acept=false"></script>

Substitua o valor clientId pelo ID do cliente fornecido ao criar sua conta de pagamento.

4. Configuração do Botão

Se o SDK requer configuração adicional, siga as instruções fornecidas pelo serviço de pagamento para configurar o botão. Isso pode incluir a definição de parâmetros adicionais ou a execução de funções de inicialização.

5. Adicionar Estilos (Opcional)

Para ajustar a aparência do botão de pagamento, você pode adicionar estilos CSS adicionais. Por exemplo:

#void-button-container {
    margin-top: 20px;
}

button {
    background-color: #0a0a0a;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    text-align: center;
}

6. Teste a Integração

Antes de colocar a integração em produção, teste o botão de pagamento em um ambiente de desenvolvimento para garantir que tudo está funcionando corretamente. Verifique se o botão aparece na página e se o processo de pagamento é iniciado corretamente quando clicado.

Funcionalidade do Botão de Pagamento

Quando o usuário clica no botão de pagamento, o SDK do serviço de pagamento irá:

  1. Inicializar o Pagamento: Iniciar o processo de pagamento com base no token do botão e nas configurações fornecidas.
  2. Exibir a Interface de Pagamento: Mostrar a interface de pagamento ao usuário para inserir detalhes de pagamento.
  3. Processar o Pagamento: Enviar os detalhes de pagamento para o serviço e processar a transação.
  4. Redirecionar ou Exibir Confirmação: Após o pagamento, o usuário será redirecionado para uma página de sucesso ou receberá uma confirmação do pagamento, dependendo da configuração.

Resolução de Problemas

Se você encontrar problemas durante a integração, verifique o seguinte:

  • Console do Navegador: Abra o console do navegador para verificar se há erros de JavaScript.
  • Verifique Tokens: Certifique-se de que os tokens do botão estão corretos e que correspondem à sua conta de pagamento.
  • Documentação do SDK: Consulte a documentação do SDK do serviço de pagamento para instruções adicionais e requisitos de configuração.

Contato para Suporte

Se precisar de mais assistência, entre em contato com o suporte técnico do serviço de pagamento ou consulte a documentação oficial fornecida.
